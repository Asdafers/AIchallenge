# Google for Startups AI Agent Challenge

## Core Opportunity

The challenge asks startups to build autonomous systems that move from prototype to production and drive measurable business outcomes. Eligible startups receive Google Cloud credits and can compete for cash, credits, expert access, event tickets, and social promotion.

## Deadline

Submissions are due by 5:00 PM PT on June 5, 2026.

## Tracks

### Track 1: Build - Net-New Agents

Build a new autonomous agent for a complex business problem. The project should show how the agent moves beyond static code and chat-style responses into declarative intent, external tool use, context gathering, and autonomous task execution.

Relevant technologies:

- Agent Development Kit (ADK), or a supported open-source framework such as LangChain or CrewAI.
- Model Context Protocol (MCP) for secure external tool and data access.
- Gemini API or another supported model path on Google's agent platform.
- Google Cloud deployment, such as Cloud Run or GKE.

### Track 2: Optimize - Existing Agents

Bring an existing experimental agent and improve its reliability. The emphasis is rigorous AI quality engineering: stress testing, multi-step reasoning evaluation, observability, debugging stalled logic, and refining system instructions until the agent can handle real-world edge cases.

### Track 3: Refactor - Google Cloud Marketplace and Gemini Enterprise

Refactor an existing working agent into a Google Cloud ecosystem-ready product. This track focuses on B2B use cases, Google Cloud-native runtime, Gemini-powered reasoning, Agent Identity, Agent-to-Agent interoperability, and readiness for Google Cloud Marketplace or Gemini Enterprise.

## Judging Criteria

- Technical Implementation: 30%
- Business Case: 30%
- Innovation and Creativity: 20%
- Demo and Presentation: 20%

## Prize Summary

- Overall Grand Prize: 1 winner, $15,000 cash and $10,000 Google Cloud credits.
- Best of Each Theme: 3 winners, $10,000 cash and $7,500 Google Cloud credits each.
- Regional Winners: 2 winners across APAC and EMEA, $5,000 cash and $2,500 Google Cloud credits each.

Additional prize benefits include virtual coffee with Addy Osmani and Google Cloud experts, VIP tickets to a Google event in the Bay Area, and social promotion.
